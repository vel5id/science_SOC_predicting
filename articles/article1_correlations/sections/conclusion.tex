\section{Заключение}
\label{sec:conclusion}
% ============================================================

Корреляционный анализ 1085 образцов и 530 признаков ДЗЗ позволил установить иерархию предсказуемости почвенных свойств Северного Казахстана по данным дистанционного зондирования.

Наиболее перспективным для спутникового картирования оказался pH: три ведущих предиктора~--- спектральный (GNDVI, $|\rho| = 0.67$), климатический (MAP, $|\rho| = 0.66$) и топографический (уклон, $|\rho| = 0.61$)~--- относятся к разным группам и, по-видимому, частично независимы; при этом 82\% корреляции pH--GNDVI сохраняется после контроля MAP и широты. K$_2$O отличается минимальной чувствительностью к пространственному конфаундингу ($\Delta < 5\%$ после контроля широты), что, вероятно, указывает на локальный минералогический контроль. Положение SOC существенно скромнее: обнаруженный конфаундинг~--- 42\% корреляции SOC--NDVI опосредовано pH~--- служит предостережением для работ, использующих вегетационные индексы как предикторы органического углерода без контроля кислотности.

NO$_3$ демонстрирует выраженную сезонную зависимость (в осенних данных $|\rho|$ с NDVI достигает 0.81, хотя подвыборка $n = 54$ недостаточна для окончательных выводов), тогда как S оказалась по существу непредсказуемой из мультиспектральных данных: декластеризация снижает Moran~$I$ с 0.77 до 0.15, ICC составляет лишь 0.17.

Высокие значения Moran~$I$ (0.51--0.86) и значительные range семивариограмм (до 137~км для pH) делают обязательным применение пространственных стратегий валидации~--- таких как Field-LOFO-CV и Farm-LOFO-CV~--- при построении прогностических моделей. Установленная иерархия, вместе с оценками конфаундинга и эффективного размера выборки, может служить отправной точкой для предиктивного моделирования (Часть~2), где запланированы расчёт корреляций по годовым подвыборкам, нелинейный анализ, интеграция данных об удобрениях и использование гиперспектральных данных для SOC.
