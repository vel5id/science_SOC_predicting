\section{Заключение}
\label{sec:conclusion}
% ============================================================

В настоящей работе проведено систематическое сравнение 11~ML- и 4~DL-моделей для предсказания шести агрохимических свойств почвы (pH, SOC, NO$_3$, P$_2$O$_5$, K$_2$O, S) по мультиспектральным спутниковым данным (Sentinel-2, Landsat~8, Sentinel-1) с трёхуровневой иерархической валидацией.
Основные результаты:

\begin{enumerate}[nosep]
  \item \textbf{Ансамблевые модели (RF, GBDT, ET) превосходят все альтернативы} при табличном моделировании.
        GBDT достигает $\rho = 0.857$ для pH, RF~--- $\rho = 0.775$ для NO$_3$ (Field-LOFO-CV).
        Разрыв с одиночным деревом (CART) составляет до $\Delta\rho = 0.50$ (SOC).

  \item \textbf{CNN конкурентоспособны только для отдельных свойств.}
        Ablation study показал, что CNN с патчами 32$\times$32 + NDVI достигает $R^2 = 0.878$ для pH, но уступает ансамблям для SOC и NO$_3$.

  \item \textbf{Мультисезонный ConvNeXt с SE-блоками обеспечивает +36\% прирост $R^2$ для NO$_3$}
        ($0.422 \to 0.575$), но переобучается для P$_2$O$_5$ ($R^2 = -0.221$).
        Мультисезонные данные наиболее полезны для мобильных элементов.

  \item \textbf{Иерархическая валидация (11~моделей $\times$ 3~стратегии) выявляет масштаб пространственной утечки:}
        при переходе от Field-LOFO к Farm-LOFO $\rho$ для pH падает на 6\% (RF: $0.798 \to 0.750$), для SOC~--- на 28\% (RF: $0.731 \to 0.529$), для NO$_3$~--- на 70\% (RF: $0.775 \to 0.232$).
        Farm-LOFO для всех 11~моделей подтвердил, что масштаб падения определяется структурой данных, а не алгоритмом.

  \item \textbf{Сера (S) трудно предсказуема из спутниковых данных}: $\rho = 0.289$ (XGBoost, Farm-LOFO) при артефактно завышенном $R^2 = 0.629$ из-за скошенного распределения; аудит выявил temporal leakage.

  \item \textbf{Для оперативного применения рекомендуются RF и CatBoost} на 15~отобранных признаках с калибровкой на каждом хозяйстве.
        Для экстраполяции на новые территории pH ($\rho = 0.750$, RF, Farm-LOFO) и P$_2$O$_5$ ($\rho = 0.571$, SVR) показывают приемлемую предсказуемость; SOC ($\rho = 0.554$, CatBoost)~--- умеренную.
\end{enumerate}

\textbf{Научная ценность исследования.}
Данная работа вносит следующий вклад в направление ЦПК:
\begin{itemize}[nosep]
  \item Представлен первый комплексный бенчмарк 15~алгоритмов (11~ML + 4~DL) для шести агрохимических свойств почвы в условиях аридной степи Центральной Азии~--- региона, ранее не охваченного систематическими DSM-исследованиями.
  \item Предложена трёхуровневая иерархическая схема валидации (Field-LOFO $\to$ Farm-LOFO $\to$ per-fold~MDI), позволяющая количественно разделить вклад пространственной утечки и утечки через отбор признаков~--- методология, применимая к любым пространственным ML-задачам.
  \item Количественно показано, что локальные модели превосходят глобальный продукт SoilGrids~v2.0 в~$3.6\times$ по~$\rho$ для pH~--- обосновывая необходимость региональных моделей высокого разрешения для точного земледелия.
  \item Физико-химические данные ($n = 1085$) и полный код экспериментов опубликованы в открытом доступе, что создаёт основу для воспроизводимых исследований и межрегиональных сравнений.
\end{itemize}

\textbf{Направления дальнейших исследований:}
(a)~transfer learning с предобученными спутниковыми фундациями (SSL4EO-S12, SatMAE, GFM) для преодоления ограничения малых выборок;
(b)~включение временны\'{x} SAR-признаков (Sentinel-1 coherence, интерферометрическая фаза) для улучшения предсказания влажностно-зависимых свойств;
(c)~расширение набора целевых свойств за счёт \textbf{тяжёлых металлов} (Cd, Pb, Zn, Cu, As)~--- анализ корреляций между спутниковыми ковариатами и содержанием тяжёлых металлов в почве представляет важное направление для экологического мониторинга и оценки пригодности сельскохозяйственных угодий;
(d)~использование \textbf{гиперспектральных данных} (миссии PRISMA, EnMAP, EMIT) с сотнями спектральных каналов для повышения точности предсказания SOC и тяжёлых металлов~--- узкие полосы поглощения в SWIR-диапазоне (2100--2350~нм), недоступные мультиспектральным сенсорам, способны значительно улучшить предиктивную способность моделей;
(e)~расширение выборки на сопредельные почвенно-климатические зоны (лесостепь, степь) для оценки переносимости моделей;
(f)~генерация пространственных карт предсказанных свойств и их верификация полевыми данными.

\medskip
\noindent Данное исследование формирует целостный методический фреймворк~--- от статистического анализа пространственных зависимостей до построения предиктивных моделей с многоуровневым аудитом утечек~--- для цифрового почвенного картирования аридных и полуаридных агроэкосистем Центральной Азии. Открытая публикация физико-химических данных и кода обеспечивает полную воспроизводимость и создаёт основу для межрегиональных сравнительных исследований.

\section*{Доступность данных и кода}

Спутниковые данные получены из открытых источников: Sentinel-2 и Sentinel-1 (Copernicus Open Access Hub), Landsat-8 (USGS EarthExplorer), SRTM DEM (NASA), SoilGrids~v2.0 (ISRIC), ERA5-Land (Copernicus CDS).
